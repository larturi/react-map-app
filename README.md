# Mapbox React App

### PoC de [Mapbox API](https://www.mapbox.com/) con React & Sockets.io

## 🚀 Quick start

## Getting Started

Install depencences:
```bash
yarn install
```

First, run the development server:

```bash
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
